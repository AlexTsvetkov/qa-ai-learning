# Module 4: LLM Behavior & Quality Testing

## 📋 Module Overview
The LLM (Llama 3 via vLLM) generates the final responses users see. This module teaches you to test for hallucinations, evaluate response quality, test prompt effectiveness, and understand LLM-specific failure modes.

## 🎯 Learning Objectives
- Detect and categorize hallucinations
- Evaluate response quality using structured criteria
- Test system prompt effectiveness and boundaries
- Understand LLM-specific bugs (token limits, temperature effects, format issues)
- Test the LLM Gateway and /chat, /embed, /ocr endpoints
