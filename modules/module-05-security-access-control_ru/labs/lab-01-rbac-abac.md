# Лаб 1: Тестирование RBAC/ABAC

## 🎯 Цель
Систематически протестировать ролевой и атрибутный контроль доступа на платформе.

## ⏱️ Время: 2 часа

## Требования
- Тестовые аккаунты с разными ролями (Сотрудник, Менеджер, Руководитель, HR, Финансы)
- Доступ к Langfuse для проверки retrieval-результатов

## Упражнение 1: RBAC — доступ к приложениям (30 мин)

| Роль | HR App | Finance App | Clerk App |
|------|--------|-------------|-----------|
| Сотрудник | Ожидание: ✅ Факт: | Ожидание: ❌ Факт: | Ожидание: ✅ Факт: |
| HR Менеджер | Ожидание: ✅ Факт: | Ожидание: ❌ Факт: | Ожидание: ❌ Факт: |
| Финансовый аналитик | Ожидание: ❌ Факт: | Ожидание: ✅ Факт: | Ожидание: ❌ Факт: |
| Руководитель | Ожидание: ✅ Факт: | Ожидание: ✅ Факт: | Ожидание: ✅ Факт: |

## Упражнение 2: ABAC — доступ к документам (45 мин)

| # | Пользователь | Вопрос | Доступ | Ответ содержит закрытые данные? | Langfuse: какие документы найдены |
|---|-------------|--------|--------|-------------------------------|----------------------------------|
| 1 | Сотрудник | «Зарплатные вилки?» | Запрещён | | |
| 2 | Менеджер | «Зарплатные вилки?» | Разрешён | | |
| 3 | Сотрудник | «Компенсация руководства?» | Запрещён | | |

**Критично:** для каждого случая «Запрещён» проверьте Langfuse трейс — были ли закрытые документы извлечены и затем отфильтрованы?

## Упражнение 3: Пропагация смены роли (30 мин)
Сменили роль → проверили сразу → через 5 мин → через 30 мин. Когда обновились права?
