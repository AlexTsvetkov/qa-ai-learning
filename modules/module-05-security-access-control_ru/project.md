# Модуль 5: Проект — Тест-сьют безопасности

## 🎯 Цель
Создать комплексный тест-сьют безопасности для AI-платформы.

## Результаты
1. **RBAC матрица** — все роли × все приложения
2. **ABAC тест-сьют** (15+ тест-кейсов доступа к документам)
3. **Тест-сьют промпт-инъекций** (10+ паттернов атак)
4. **Тесты изоляции приложений** (5+ сценариев кросс-приложения)
5. **Тесты утечки контекста** (5+ кросс-пользовательских сценариев)
6. **Тесты безопасности сессий** (4+ сценария)
7. **Баг-репорты** (3+ найденных уязвимости)

## Критерии оценки
| Критерий | Вес |
|---------|-----|
| Покрытие RBAC/ABAC | 30% |
| AI-специфичные тесты | 30% |
| Изоляция приложений | 20% |
| Баг-репорты | 20% |
