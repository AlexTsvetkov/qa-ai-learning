# Модуль 5: Безопасность и контроль доступа

## 📋 Обзор модуля
Безопасность AI-платформы выходит за рамки традиционного тестирования. RBAC/ABAC контролирует, кто какие документы видит, но AI создаёт новые вектора атак: утечка контекста через ответы LLM, промпт-инъекции, кросс-приложение.

## 🎯 Цели обучения
- Тестировать RBAC/ABAC контроль доступа на каждом уровне
- Обнаруживать утечку данных через AI-ответы
- Тестировать изоляцию между приложениями
- Выявлять AI-специфичные уязвимости
- Проверять интеграцию с Active Directory
