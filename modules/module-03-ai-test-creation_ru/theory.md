# Модуль 3: Теория — Создание тестов с AI

## Генерация тест-кейсов
- Из требований и user stories
- Исследовательские чартеры
- Тестовые матрицы (pairwise)

## Тестовые данные
- По схеме (schema-based)
- По сценариям
- Adversarial данные (XSS, SQL injection)

## Автоматизация
- Конвертация ручных тестов в автотесты (Playwright, pytest)
- Page Object Model

## Ревью тестов с AI
- Поиск пробелов в покрытии
- Улучшение assertions
- Устранение flakiness

**Всегда проверяйте AI-сгенерированные тесты!**
