# Модуль 7: Квиз — Тест-стратегия и онбординг

### В1: Какие две области тестирования должны быть P0 (наивысший приоритет)?
**Ответ:** Обнаружение галлюцинаций LLM и проверка RBAC/ABAC — это P0, так как имеют критическое влияние.

### В2: «Ответ AI выглядит правильным, поэтому проверять Langfuse трейс не нужно.» Верно?
**Ответ:** Анти-паттерн. Ответ может быть правильным случайно, при этом внутренний поток может быть сломан (не те документы, не тот инструмент).

### В3: К дню 7 онбординга новый QA должен уметь:
**Ответ:** Обнаруживать галлюцинации, тестировать RBAC, использовать фреймворк GRACE для оценки качества.

### В4: Как часто нужно запускать Golden Set регрессию?
**Ответ:** При каждом деплое / билде, чтобы ловить регрессии.

### В5: Компонент с НИЗКОЙ вероятностью отказа, но КРИТИЧЕСКИМ влиянием:
**Ответ:** Тестировать регулярно — критическое влияние важнее низкой вероятности.
