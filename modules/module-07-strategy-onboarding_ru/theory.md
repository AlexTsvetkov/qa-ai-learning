# Модуль 7: Теория — Тест-стратегия, онбординг и роадмэп

## 7.1 Тест-стратегия

### Уровни тестирования
1. UI (React) 2. API (.NET) 3. Агент (Agent Runtime, MCP) 4. Качество AI (LLM, галлюцинации, RAG)
5. Данные (Ingestion, KB, Weaviate) 6. Безопасность (RBAC/ABAC) 7. Инфраструктура (vLLM, GPU, Redis)

### Матрица рисков
| Компонент | Вероятность | Влияние | Приоритет |
|-----------|-----------|---------|-----------|
| LLM галлюцинации | Высокая | Critical | P0 |
| RBAC/ABAC утечка | Средняя | Critical | P0 |
| RAG качество retrieval | Средняя | High | P1 |
| Выбор инструмента агентом | Средняя | High | P1 |
| Ingestion pipeline | Средняя | Medium | P2 |

### Критические сценарии (must-pass)
1. E2E happy path 2. Golden set без галлюцинаций 3. RBAC enforcement 4. Кросс-приложение изоляция
5. Выбор инструментов 6. Промпт-инъекции заблокированы 7. Ingestion 8. Baseline латентности

### Анти-паттерны
- ❌ Доверять ответам AI без проверки
- ❌ Тестировать только happy path
- ❌ Игнорировать Langfuse трейсы
- ❌ Тестировать с одной ролью
- ❌ Пропускать регрессию после смены промпта

## 7.2 14-дневный план онбординга

| Дни | Тема | Результат |
|-----|------|-----------|
| 1-2 | Архитектура (Модуль 1) | Карта потока данных, шпаргалка |
| 3-4 | Агент и MCP (Модуль 2) | Тесты выбора инструментов |
| 5-6 | RAG и KB (Модуль 3) | Golden set, тесты ingestion |
| 7-8 | LLM и Безопасность (Модули 4-5) | Отчёт о галлюцинациях, RBAC тесты |
| 9-10 | Наблюдаемость (Модуль 6) | Playbook отладки |
| 11-14 | Стратегия и самостоятельная работа | Тест-стратегия, финальный проект |

**К дню 14 QA-инженер должен:** понимать архитектуру, тестировать любой компонент, отлаживать через Langfuse/Grafana, иметь golden set, работать самостоятельно.

## 7.3 Роадмэп обучения

| Фаза | Недели | Цель | Критерий выхода |
|------|--------|------|----------------|
| Foundation | 1-2 | Понять систему | Нарисовать data flow, найти трейс |
| Core | 3-4 | RAG и LLM тестирование | Golden set, обнаружение галлюцинаций |
| Advanced | 5-6 | Безопасность, отладка | RBAC тесты, debugging workflow |
| Production | 7-8 | Самостоятельная работа | Тест-стратегия, онбординг других |
