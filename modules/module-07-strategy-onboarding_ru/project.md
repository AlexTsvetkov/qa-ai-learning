# Модуль 7: Финальный проект — QA-академия

## 🎯 Цель
Продемонстрировать мастерство, создав полный набор QA-артефактов.

## ⏱️ Время: 6 часов

## Результаты

### 1. Полная тест-стратегия
7 уровней, 15+ рисков, 10+ критических сценариев, AI-специфичный подход, анти-паттерны, процесс релиза.

### 2. 14-дневный план онбординга (кастомизированный)
Подневной план, чеклисты, гид по первым багам (10+), критерии входа/выхода, список доступов.

### 3. Golden Set тест-сьют (30+ записей)
Пары вопрос-документ по всем категориям, негативные тесты, кросс-ролевые тесты.

### 4. Playbook отладки
5+ типов проблем с пошаговыми гидами, справочник по Langfuse и Grafana.

### 5. Портфолио багов (10+ из всех модулей)
Баги за весь курс, категоризированные по компоненту и критичности, с доказательствами из трейсов.
