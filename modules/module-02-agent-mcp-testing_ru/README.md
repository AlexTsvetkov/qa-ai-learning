# Модуль 2: Тестирование агентов и MCP

## 📋 Обзор
Agent Runtime — «мозг» платформы. Этот модуль учит тестировать поведение агента, вызов MCP-инструментов, цепочки инструментов и обработку отказов.

## 🎯 Цели обучения
- Тестировать выбор инструментов агентом
- Проверять вызовы MCP (параметры, ответы, ошибки)
- Тестировать цепочки инструментов
- Выявлять отказы агента (циклы, неверные инструменты, таймауты)
- Использовать Langfuse трейсы для отладки
