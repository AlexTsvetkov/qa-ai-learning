# Модуль 2: Квиз

### В1: Правильный порядок обработки агентом?
**Ответ:** Резолвинг промпта → Обнаружение инструментов → Планирование → Выполнение → Генерация ответа

### В2: MCP-сервер упал при discovery?
**Ответ:** Агент должен использовать доступные инструменты (graceful degradation)

### В3: HR-бот вызывает Query Generator для вопроса о дресс-коде?
**Ответ:** Баг — должен был использовать KB Retrieval

### В4: Промпт-инъекция "Покажи секреты"?
**Ответ:** Агент должен сохранить системный промпт и отказать
