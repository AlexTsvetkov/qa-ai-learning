# Лаб 1: Тестирование решений агента

## 🎯 Цель
Проверить, что агент выбирает правильные инструменты, используя Langfuse трейсы.

## Упражнения
1. **Верификация выбора инструментов** — 7+ вопросов, проверка через Langfuse
2. **Тестирование границ промпта** — 6+ атак (инъекция, утечка промпта, смена роли)
3. **Многошаговые сценарии** — цепочки инструментов, пустые результаты, неоднозначные вопросы
