# Module 5: Project — AI-Enhanced Testing Pipeline

## 🎯 Objective
Build a complete AI-enhanced testing pipeline for a sample project.

## ⏱️ Время: 4 часов

---

## Результаты

### 1. AI Test Helper Library (`scripts/ai_test_helper.py`)
A reusable Python module with functions for:
- Test failure analysis
- Test data generation
- Test suggestion from code diffs
- Test report generation

### 2. pytest Integration (`conftest.py`)
- Auto-analyze failures
- AI-powered test data fixtures
- Smart test summary generation

### 3. CI/CD Workflow (`.github/workflows/ai-testing.yml`)
- Triggered on PRs
- AI test suggestions for changed code
- AI failure analysis on test failures
- AI-powered test report as PR comment

### 4. Documentation (`exercises/module05_pipeline_docs.md`)
- Architecture diagram
- Setup instructions
- Cost estimation for your pipeline
- Known limitations
