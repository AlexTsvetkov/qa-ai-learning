# Лаб 1: Анализ трейсов Langfuse и отладка

## 🎯 Цель
Научиться свободно читать Langfuse трейсы для отладки проблем AI-платформы.

## ⏱️ Время: 2 часа

## Упражнение 2: Отладка неверного ответа (45 мин)

1. Найдите вопрос, где AI даёт неверный ответ
2. Откройте Langfuse трейс
3. Следуйте workflow отладки:

| Шаг | Проверяем | Результат |
|-----|----------|-----------|
| Retrieval | Правильные документы? Скоры? | |
| LLM Input | Контекст полный? Версия промпта? | |
| LLM Output | Точно ли LLM передал контекст? | |
