# Модуль 6: Проект — Playbook мониторинга Production

## 🎯 Цель
Создать полный playbook мониторинга и отладки для QA-команды.

## Результаты

### 1. Playbook отладки
Пошаговые гиды для 5 самых частых проблем:
- Неверный ответ (галлюцинация)
- Медленный ответ
- Пустой ответ / ошибка
- Отказ контроля доступа
- Устаревшая информация

Каждый гид: симптомы → проверка Langfuse → проверка Grafana → проверка логов → решение.

Идеальный QA-дашборд: панели, метрики, пороги, правила алертов.

### 3. Pre/post-release чеклисты
Полные чеклисты для проверки до и после каждого деплоя.

Шаблон для документирования и эскалации инцидентов AI-платформы.
