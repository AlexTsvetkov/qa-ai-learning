# Модуль 6: Квиз — Наблюдаемость и Production QA

### В1: Первый шаг при отладке неверного ответа?
**Ответ:** Найти Langfuse трейс для этого запроса.

### В2: В Langfuse трейсе спан `mcp_retrieval` вернул 0 результатов. Что это значит?
**Ответ:** База знаний не нашла релевантных документов. LLM должен сказать «У меня нет информации», а НЕ галлюцинировать.

### В3: Grafana показывает LLM латентность p95 = 25с (норма <10с), GPU память 95%. Причина?
**Ответ:** vLLM перегружен — недостаточно GPU-памяти при текущей нагрузке.

### В4: Перед деплоем новой версии промпта нужно:
**Ответ:** Запустить Golden Set регрессионный тест и сравнить результаты с предыдущей версией.

### В5: Langfuse трейсы лучше для _____, а Prometheus метрики лучше для _____:
**Ответ:** Трейсы — для отладки отдельных запросов; метрики — для мониторинга системных трендов.
