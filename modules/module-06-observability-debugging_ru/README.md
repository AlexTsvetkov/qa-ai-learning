# Модуль 6: Наблюдаемость и Production QA

## 📋 Обзор модуля
Наблюдаемость — ваш основной набор инструментов для отладки. Этот модуль учит использовать Langfuse трейсы, Prometheus метрики, Grafana дашборды и логи приложений для отладки AI-специфичных проблем и мониторинга качества в продакшене.

## 🎯 Цели обучения
- Читать и интерпретировать Langfuse трейсы для отладки AI-поведения
- Использовать Prometheus метрики для выявления проблем производительности
- Читать Grafana дашборды для QA-мониторинга
- Отлаживать production-проблемы по структурированным логам
- Создавать чеклисты мониторинга для релизов
