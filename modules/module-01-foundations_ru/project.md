# Модуль 1: Проект — Исследование AI-инструментов для QA

## 🎯 Цель
Оценить минимум 2 AI-инструмента для QA и создать сравнительный отчёт.

## Задание
1. Выберите функциональность для тестирования
2. Задайте каждому AI 5 одинаковых вопросов (тест-кейсы, безопасность, баг-репорт, данные, ревью)
3. Создайте сравнительный отчёт в `exercises/module01_project_report.md`
