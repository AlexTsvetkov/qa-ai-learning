# Лабораторная 1: Настройка рабочего окружения

## 🎯 Цель
Настроить инструменты для работы с AI в QA.

## Шаги
1. Установите Python 3.10+
2. Клонируйте репозиторий, создайте venv
3. `pip install openai anthropic requests pytest`
4. Создайте аккаунты OpenAI и Anthropic
5. Настройте `.env` с API-ключами

> ⚠️ НИКОГДА не коммитьте API-ключи в Git!
