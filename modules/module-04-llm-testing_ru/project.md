# Модуль 4: Проект — Оценка качества LLM

## 🎯 Цель
Создать комплексный фреймворк оценки качества LLM для платформы.

## Результаты
1. Тест-сьют галлюцинаций (15+ кейсов): известные ответы, ловушки, консистентность
2. GRACE-отчёт (10+ оценок) по всем приложениям
3. Тест-сьют системных промптов (8+ кейсов)
4. Тесты LLM-эндпоинтов (/chat, /embed, /ocr)
5. Баг-репорты (3+ бага)

## Критерии оценки
| Критерий | Вес |
|---------|-----|
| Обнаружение галлюцинаций | 30% |
| Качество GRACE-оценок | 25% |
| Тестирование промптов | 20% |
| Тесты эндпоинтов | 15% |
| Баг-репорты | 10% |
