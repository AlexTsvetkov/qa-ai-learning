# Лаб 1: Обнаружение и классификация галлюцинаций

## 🎯 Цель
Научиться систематически находить, классифицировать и документировать галлюцинации в AI-ответах.

| 3-10 | ... | | | | |

**Типы:** Фабрикация, Противоречие, Экстраполяция, Ошибка атрибуции, Конфляция
