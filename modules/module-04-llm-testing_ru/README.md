# Модуль 4: Тестирование поведения LLM

## 📋 Обзор модуля
LLM (Llama 3 через vLLM) генерирует финальные ответы, которые видят пользователи. Этот модуль учит обнаруживать галлюцинации, оценивать качество ответов, тестировать эффективность промптов и понимать LLM-специфичные отказы.

## 🎯 Цели обучения
- Обнаруживать и классифицировать галлюцинации
- Оценивать качество ответов по фреймворку GRACE
- Тестировать системные промпты и их границы
- Понимать LLM-специфичные баги (лимиты токенов, температура, формат)
- Тестировать LLM Gateway эндпоинты /chat, /embed, /ocr
