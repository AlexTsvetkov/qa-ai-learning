# Module 6: Project — Autonomous Testing Prototype

## 🎯 Objective
Build a working prototype that combines self-healing tests with an AI testing agent.

## ⏱️ Estimated Time: 5 hours

---

## Deliverables

### 1. Self-Healing Test Framework (`exercises/framework/`)
- `self_healing.py` — SelfHealingLocator implementation
- `page_objects.py` — Page Objects using self-healing locators
- `test_with_healing.py` — Tests demonstrating self-healing
- `healing_report.py` — Report generator

### 2. AI Testing Agent (`exercises/agent/`)
- `agent.py` — Complete AITestingAgent implementation
- `agent_config.json` — Configuration (URLs, max steps, etc.)
- `run_agent.py` — Script to run the agent

### 3. Findings Report (`exercises/module06_report.md`)
- Self-healing test results (how many healed, success rate)
- Agent exploration results (pages visited, issues found)
- Effectiveness analysis (real bugs found vs false positives)
- Recommendations for improvement
- Reflection on the future of AI in QA

### 4. Demo Video/Screenshots
- Screenshots or screen recording showing the agent in action
- Before/after of self-healing locator changes

## Evaluation Criteria

| Criteria | Points |
|----------|--------|
| Self-healing framework quality | 25 |
| AI agent implementation | 25 |
| Findings report depth | 20 |
| Demo quality | 15 |
| Reflection and recommendations | 15 |
| **Total** | **100** |
