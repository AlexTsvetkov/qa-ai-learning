# Module 6: Quiz — Autonomous Testing & Future

---

### Q1: What is a "self-healing" test?
a) A test that fixes application bugs  
b) A test that automatically adapts when UI elements change  
c) A test that runs itself on schedule  
d) A test that recovers from network failures  

### Q2: What is the main purpose of an AI testing agent?
a) To replace all manual testers  
b) To autonomously explore and find potential issues in an application  
c) To write all test code automatically  
d) To deploy applications  

### Q3: Which visual testing approach has the fewest false positives?
a) Pixel-by-pixel comparison  
b) Perceptual diff  
c) AI-powered comparison  
d) Manual visual review  

### Q4: What is a key ethical consideration when using AI in testing?
a) AI should make all quality decisions  
b) Humans should be accountable and validate AI recommendations  
c) AI bias is not a concern in testing  
d) Transparency is optional  

### Q5: What is the "observe-decide-execute-evaluate" loop?
a) A software development lifecycle  
b) The core cycle of an AI testing agent  
c) A bug triage process  
d) A CI/CD pipeline pattern  

### Q6: Which future trend involves AI predicting where bugs will occur?
a) Self-healing tests  
b) Visual AI testing  
c) Predictive quality  
d) Autonomous deployment  

---

<details>
<summary>Нажмите для ответов</summary>

1. **b)** A test that automatically adapts when UI elements change
2. **b)** To autonomously explore and find potential issues
3. **c)** AI-powered comparison — it understands context and intent
4. **b)** Humans should be accountable and validate AI recommendations
5. **b)** The core cycle of an AI testing agent
6. **c)** Predictive quality — using ML to predict bug-prone areas
