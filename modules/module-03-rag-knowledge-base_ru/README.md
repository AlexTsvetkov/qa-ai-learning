# Модуль 3: Тестирование RAG и базы знаний

## 📋 Обзор модуля
База знаний — основа точных AI-ответов. Этот модуль покрывает полный RAG-пайплайн: от загрузки документов через поиск до оценки качества ответов. Вы научитесь тестировать качество retrieval, ingestion-пайплайн, чанкинг и векторную БД (Weaviate).

## 🎯 Цели обучения
- Тестировать качество retrieval (правильные документы для правильных вопросов)
- Проверять ingestion-пайплайн (документ → чанки → эмбеддинги → Weaviate)
- Тестировать качество чанкинга и его влияние на ответы
- Выявлять причины ошибок retrieval
- Тестировать Knowledge Base REST API напрямую
