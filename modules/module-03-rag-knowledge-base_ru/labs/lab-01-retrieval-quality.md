# Лаб 1: Тестирование качества retrieval

## 🎯 Цель
Проверить, что база знаний возвращает правильные документы с правильным ранжированием.

## ⏱️ Время: 2 часа

## Упражнение 1: Создание Golden Set (45 мин)

Создайте минимум 10 пар вопрос-документ, где вы ЗНАЕТЕ правильный ответ:

| # | Вопрос | Ожидаемый документ | Раздел/Страница | Релевантность |
|---|--------|-------------------|----------------|--------------|
| 1 | «Какова политика отпусков?» | | | Высокая |
| 2 | «Сколько больничных в год?» | | | Высокая |
| 3-10 | ... | | | |

Для каждого вопроса запросите KB API и запишите: найден ли документ, правильный ли, позиция в ранжировании, скор релевантности.

**Критерий прохождения:** правильный документ в топ-3 с релевантностью > 0.7

## Упражнение 2: Семантическое сходство (30 мин)

Разные формулировки одного вопроса должны возвращать те же документы:

| Оригинал | Перефразировка | Тот же топ-результат? | Разница скоров |
|---------|---------------|---------------------|---------------|
| «политика отпусков» | «правила отгулов» | | |
| «политика отпусков» | «сколько дней отдыха» | | |
| «декретный отпуск» | «отпуск по рождению ребёнка» | | |

## Упражнение 3: Негативное тестирование (30 мин)

Нерелевантные вопросы должны давать пустые или низкорелевантные результаты:

| Вопрос | Ожидание | Результат | Макс. скор | Пройден? |
|--------|----------|-----------|-----------|---------|
| «квантовая физика» | Нет релевантных | | | |
| «рецепт шоколадного торта» | Нет релевантных | | | |
| Случайные символы | Нет релевантных | | | |

## Упражнение 4: Качество ранжирования (15 мин)

Для широкого запроса проверьте порядок результатов и оцените, правильно ли ранжированы.
