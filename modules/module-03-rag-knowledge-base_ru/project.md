# Модуль 3: Проект — RAG тест-сьют

## 🎯 Цель
Создать комплексный фреймворк тестирования RAG для платформы.

## ⏱️ Время: 3 часа

## Результаты

### 1. Golden Set (20+ пар вопрос-документ)
Покрытие всех категорий документов в базе знаний. Каждая запись: вопрос, ожидаемый документ, раздел, минимальный скор релевантности.

### 2. Тест-план ingestion (12+ тест-кейсов)
Обычные файлы, граничные случаи (большой, битый, пустой, Unicode), дубликаты, обновление, OCR-качество.

### 3. Отчёт о качестве retrieval
Запуск Golden Set: precision, recall, средний скор, качество ранжирования, замеры латентности.

### 4. Анализ качества чанкинга
Инспекция 10+ чанков в Weaviate: распределение размеров, качество границ, перекрытие, обработка таблиц.

### 5. Баг-репорты (3+ бага)

## Критерии оценки
| Критерий | Вес |
|---------|-----|
| Полнота Golden Set | 30% |
| Покрытие ingestion | 25% |
| Глубина анализа retrieval | 25% |
| Качество баг-репортов | 20% |
