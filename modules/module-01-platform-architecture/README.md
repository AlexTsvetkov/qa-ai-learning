# Module 1: Platform Architecture & AI Fundamentals for QA

## 📋 Module Overview

This module introduces the enterprise AI platform you'll be testing. You'll learn the complete system architecture — from user request to AI response — and understand key AI concepts (LLM, RAG, embeddings, agents) from a QA perspective. No prior AI experience needed.

## 🎯 Learning Objectives

After completing this module, you will be able to:
- Draw the complete data flow: User → FE → BE → Agent → MCP → KB → LLM → Response
- Explain what each component does (Agent Runtime, MCP, LLM Gateway, Knowledge Base, Ingestion Pipeline)
- Identify where systems typically break and what QA should test at each layer
- Understand key AI concepts (LLM, RAG, embeddings, hallucinations) without deep ML theory
- Describe the 3 platform releases and their scope (HR, Finance, Clerk)
