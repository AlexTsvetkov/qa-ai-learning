# Модуль 2: Теория — Prompt Engineering для тестировщиков

## 2.1 Что такое Prompt Engineering?
Практика создания и улучшения запросов к AI для получения наилучших результатов.

## 2.2 Основные техники
- **Zero-Shot** — задание без примеров
- **Few-Shot** — с примерами формата
- **Chain-of-Thought** — пошаговое рассуждение
- **Role-Based** — назначение роли эксперта
- **Structured Output** — указание формата вывода

## 2.3 Фреймворк CRISP
| | Элемент | Описание |
|---|---|---|
| **C** | Context | Контекст проекта |
| **R** | Role | Роль AI |
| **I** | Instructions | Задача |
| **S** | Scope | Границы |
| **P** | Pattern | Формат вывода |

- ❌ Перегрузка информацией
- ❌ Отсутствие формата
