# Module 2: Project — QA Prompt Toolkit

## 🎯 Objective

Create a comprehensive, documented prompt toolkit that your team could adopt for daily QA work.

## ⏱️ Время: 3 часов

---

## Описание задания

Create a complete prompt toolkit (`exercises/module02_prompt_toolkit.md`) that includes:

### Part 1: Prompt Templates (15 prompts)

Create **15 tested prompts** across these categories:

| Category | Required Count |
|----------|---------------|
| Test case generation | 3 |
| Bug reporting | 2 |
| Test data generation | 2 |
| Requirements analysis | 2 |
| Code review | 2 |
| Test strategy | 1 |
| API testing | 2 |
| Performance testing | 1 |

### Part 2: Usage Guide

Write a brief guide explaining:
1. How to choose the right prompting technique for a task
2. How to customize templates for specific projects
3. Common mistakes and how to avoid them

### Part 3: Before/After Examples

Show **3 examples** of:
- Original (bad) prompt
- Improved prompt using techniques from this module
- Comparison of outputs from both

---

## Результаты

- [ ] `exercises/module02_prompt_toolkit.md` with 15 prompts
- [ ] Usage guide section
- [ ] 3 before/after examples with actual AI outputs
- [ ] Each prompt tested and rated
