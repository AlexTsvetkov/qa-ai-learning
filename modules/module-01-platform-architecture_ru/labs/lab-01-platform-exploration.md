# Лаб 1: Исследование платформы и маппинг потока данных

## 🎯 Цель
Практическое исследование AI-платформы и создание карты потока данных.

## ⏱️ Время: 2 часа

## Упражнения

### 1. Проследите запрос через UI (30 мин)
- Откройте HR Ассистент, DevTools → Network
- Отправьте вопрос "Какая у нас политика отпусков?"
- Запишите: URL, HTTP метод, структуру payload, время ответа, стриминг ли ответ

### 2. Исследуйте API (30 мин)
- Найдите эндпоинты Backend API (Swagger)
- Протестируйте границы: без авторизации, пустое тело, очень длинное сообщение

### 3. Прочитайте Langfuse трейс (30 мин)
- Найдите трейс вашего запроса в Langfuse
- Определите каждый span: Agent? MCP? LLM? KB?
- Сравните трейсы для разных типов вопросов

### 4. Проверьте Grafana дашборды (30 мин)
- Запишите доступные дашборды и ключевые метрики
- Отправьте 10 быстрых запросов и наблюдайте за метриками

## 📝 Результат
Диаграмма полного потока данных с подписями компонентов и технологий.
