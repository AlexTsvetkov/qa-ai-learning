# Module 2: Quiz — Prompt Engineering for Testers

---

### Q1: What is "few-shot" prompting?
a) Asking AI to respond quickly  
b) Providing examples of desired output before the actual request  
c) Sending multiple prompts at once  
d) Using short, minimal prompts  

### Q2: What does the "C" in the CRISP framework stand for?
a) Code  
b) Context  
c) Command  
d) Completion  

### Q3: Which technique is best for complex bug analysis?
a) Zero-shot prompting  
b) Few-shot prompting  
c) Chain-of-thought prompting  
d) Structured output prompting  

### Q4: What is a prompt anti-pattern?
a) A well-tested prompt template  
b) A prompting practice that leads to poor results  
c) A pattern for structuring AI responses  
d) A type of test pattern  

### Q5: Which is a better prompt for test case generation?
a) "Write tests"  
b) "Generate 5 test cases for the login page including positive, negative, and boundary scenarios in table format"  
c) Both are equally effective  
d) Neither works with AI  

### Q6: What is role-based prompting?
a) Testing role-based access control  
b) Assigning a specific persona/expertise to the AI  
c) Prompting based on your job role  
d) Creating prompts for different team members  

### Q7: When should you use structured output prompting?
a) When you need creative, free-form responses  
b) When you need machine-readable or consistently formatted output  
c) When the task is simple and straightforward  
d) Only when using APIs  

### Q8: What is the recommended approach when your first prompt gives poor results?
a) Try a completely different AI tool  
b) Give up and do the task manually  
c) Iterate and refine the prompt  
d) Send the same prompt again  

### Q9: What does "temperature" 0 produce?
a) The most creative responses  
b) The most deterministic/consistent responses  
c) The fastest responses  
d) No response  

### Q10: Why is building a prompt library valuable?
a) It saves time by reusing tested prompts  
b) It ensures consistent quality across tasks  
c) It serves as team knowledge base  
d) All of the above  

---

## Answers

<details>
<summary>Click to reveal answers</summary>

1. **b)** Providing examples of desired output before the actual request
2. **b)** Context — background information about the project/feature
3. **c)** Chain-of-thought prompting — it encourages step-by-step reasoning
4. **b)** A prompting practice that leads to poor results
5. **b)** The detailed prompt with specific requirements and format
6. **b)** Assigning a specific persona/expertise to the AI
7. **b)** When you need machine-readable or consistently formatted output
8. **c)** Iterate and refine the prompt
9. **b)** The most deterministic/consistent responses
10. **d)** All of the above
