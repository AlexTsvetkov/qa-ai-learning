# Module 3: RAG & Knowledge Base Testing

## 📋 Module Overview
The Knowledge Base is the foundation of accurate AI responses. This module covers testing the full RAG pipeline — from document ingestion through retrieval to response quality. You'll learn to test retrieval accuracy, ingestion pipelines, chunking quality, and the vector database (Weaviate).

## 🎯 Learning Objectives
After completing this module, you will be able to:
- Test retrieval quality (right documents found for the right questions)
- Verify the ingestion pipeline (document → chunks → embeddings → Weaviate)
- Test chunking quality and its impact on answers
- Identify retrieval failures and their root causes
- Test the Knowledge Base REST API directly
