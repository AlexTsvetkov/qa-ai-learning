# Module 5: Security & Access Control Testing

## 📋 Module Overview
Security in an AI platform goes beyond traditional testing. RBAC/ABAC controls who sees what documents, but AI introduces new attack vectors: context leakage through LLM responses, prompt injection, cross-application data exposure. This module covers all security testing aspects.

## 🎯 Learning Objectives
- Test RBAC/ABAC access control at every layer
- Detect data leakage through AI responses
- Test cross-application isolation
- Identify AI-specific security vulnerabilities (prompt injection, context leakage)
- Verify Active Directory integration
