# Module 5: Project — Security Test Suite

## 🎯 Objective
Create a comprehensive security test suite for the AI platform.

## Deliverables

### 1. RBAC Test Matrix (all roles × all apps)
### 2. ABAC Test Suite (15+ document access test cases)
### 3. Prompt Injection Test Suite (10+ injection patterns)
### 4. Cross-Application Isolation Tests (5+ scenarios)
### 5. Context Leakage Tests (5+ cross-user scenarios)
### 6. Session Security Tests (4+ scenarios)
### 7. Bug Reports (3+ security bugs found)

## Evaluation Criteria
| Criteria | Weight |
|---------|--------|
| RBAC/ABAC coverage | 30% |
| AI-specific security tests | 30% |
| Cross-application isolation | 20% |
| Bug reports | 20% |
